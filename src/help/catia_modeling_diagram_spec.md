# 3D 造型 · 参数机理与图示说明（画图设计输入）

> **本文档用途**：作为绘制「3D 造型」模块 4 张示意图（1 总成 + 3 分阶段）的**设计输入**。
> 它不是给终端用户的操作手册（操作手册见 `catia_modeling_stage1/2/3.md`），
> 而是给**画图者**讲清每个参数在翼型几何上的物理含义、每张图应该画出哪些几何要素。
>
> 阅读本 md 前请先理解：三个步骤本质是对**叶片截面点云**做几何重建，
> 所有的"参数"都是在描述如何处理这些点。

---

## 0. 前置几何常识（图示的基础）

绘制任何一张图前，必须先在脑海里建立翼型截面的几何模型：

### 0.1 单个翼型截面的点云排布

一个截面（`Sect{组号}`）由 **399 个点**按顺序环绕排列，构成翼型轮廓：

```
                 前缘 LE
                 点200
                 ▲
               ╱   ╲          ← 上表面（吸力面）
             ╱       ╲
           ╱           ╲
        200            399 ●  ← 尾缘 TE（上表面端）
         │              ● 1    ← 尾缘 TE（下表面端）
           ╲           ╱
             ╲       ╱        ← 下表面（压力面）
               ╲   ╱
```

- **点序号 1 ~ 399** 沿翼型轮廓**连续环绕**：从下表面尾缘(1) → 前缘(200) → 上表面尾缘(399)
- **前缘点 LE = 点 200**（翼型最前点）
- **尾缘点 TE = 点 1 和 点 399**（翼型最后端的两个点，构成尾缘厚度）

> ⚠ 画图时**必须标注点序号**（1 / 200 / 399 及其位置），这是所有参数的基准。

### 0.2 叶片整体 = 多个截面沿展向（Z 向）堆叠

叶片不是一个截面，而是 **96 个截面**沿展向（高度方向）排列：

```
   翼尖 (Z 大，组号大，如 Sect96)
    ▲
    │  截面96 ──╮
    │  截面95 ──┤
    │   ...     │  ← 展向（Z 向）堆叠
    │  截面2  ──┤
    │  截面1  ──╯
   翼根 (Z 小，组号小，如 Sect1)
```

- 每个截面在不同高度 Z，有自己的翼型轮廓
- 三步处理的是"每个截面内的点"，最后 Loft 把它们沿展向连成曲面

## 1. 图 ① 总成图（1 张）

**目的**：一眼看清"点云 → 三步 → 叶片曲面"的整体流程与三个步骤的衔接关系。

### 必须画出的要素

| 要素 | 画法 |
|---|---|
| **输入：散乱点云** | 画一个叶片轮廓示意，用密集小圆点表示点云，标注几个 `Sect{组}_{点}` |
| **步骤① 框** | 点云 → 单个截面的样条/光顺曲线/前尾缘/弦线（截面局部） |
| **步骤② 框** | 单条曲线 → 等距重采样点 + 二次光顺（曲线局部放大） |
| **步骤③ 框** | 多条曲线 → Loft 蒙皮 → 三维曲面（等轴侧叶片） |
| **衔接数据流** | 标注每步的**输入几何集名 → 输出几何集名**（见下表） |

### 三步几何集衔接（总成图上的箭头标注）

```
点云 (Sect{组}_{点})
   │
   ├─①─▶ Z_Splines(样条) ─▶ Z_Smooths(光顺曲线)
   │                          │
   │                          └─②─▶ Z_ResamplePoints(等距点)
   │                                   Z_OriginalSpline(重拟合样条)
   │                                   Z_ResampleSmooth(二次光顺样条)
   │                                                │
   └─────────────────────────────────────────────────┴─③─▶ loft_surface(叶片曲面)
```

> **少画流程箭头**：总成图保留上面这条主数据流即可，分阶段图里不要再重复整条流程。

## 2. 图 ② 分阶段① 构建截面（点云 → 截面几何）

**目的**：讲清"一组点云如何变成一个完整截面"。

### 必须画出的要素（一个截面的放大局部图）

1. **翼型截面轮廓**：画出完整翼型形状（上下表面 + 前尾缘）
2. **点序号标注**（图示核心）：
   - 点 1（尾缘下）●—— 标 `TE₁ (点1)`
   - 点 200（前缘）●—— 标 `LE (点200)`
   - 点 399（尾缘上）●—— 标 `TE₂ (点399)`
   - 沿轮廓用虚线小点表示 1~399 的密集点云
3. **拟合样条**：用一条贯穿所有点的曲线（实线）表示 `Z_Splines`
4. **光顺曲线**：样条旁边叠一条更光滑的曲线（不同颜色），标 `Z_Smooths`
5. **前缘点 LE**：点 200 处单独画一个高亮点（亮色），标"前缘点"
6. **尾缘点 TE**：点 1、399 处各一个高亮点，标"尾缘点"
7. **弦长线**：**从前缘点 → 尾缘中点**（点1和点399的中点）画一条直线，标"弦长/弦线"
   - ⚠ **弦长不是尾缘两点连线，是前缘到尾缘中点的连线**（核对自参考脚本第 169 行）

### 该图对应的参数（图旁标注）

| 参数 | 在图上对应 |
|---|---|
| 截面数 96 | 图角注"×96 截面" |
| 起始组号 | 图角注"从 Sect{起始组} 起" |
| 点数上限 400 | 点云标注"每截面最多 399 点(序号1~399)" |
| 前缘点序号 200 | 指向图中的前缘点 ● |
| 尾缘点1 / 尾缘点2 (1, 399) | 指向图中的两个尾缘点 ● |
| 四段光顺阈值 | 用翼型四等分段着色示意（见下） |

### 四段光顺阈值的图示建议

把翼型沿点序号四等分（1~100 / 100~200 / 200~300 / 300~399），各段用不同颜色/深浅着色，标注对应阈值（4/3/2/1），表示"这四段施加不同强度的光顺"：

```
   ┌────────┬────────┬────────┬────────┐
   │ 阈值4  │ 阈值3  │ 阈值3  │ 阈值2  │   ← 四段，值越大光顺越强
   │(1~100) │(100~200│(200~300│(300~399│
   └────────┴────────┴────────┴────────┘
    尾缘下    下中段    前缘段    上中段
```

## 3. 图 ③ 分阶段② 重采样光顺（曲线 → 均匀点 + 二次光顺）

**目的**：讲清"为什么要把曲线重新采样成等距点，以及二次光顺的作用"。

### 必须画出的要素（一条曲线的局部放大，做"前后对比"）

**左右对比图（或上下对比）**：

**左侧：重采样前（步骤①的输出 Z_Smooths）**
- 一条光顺曲线
- 曲线上**不均匀**的点（前缘密、中段稀），用小圆点表示
- 标注"点分布不均 → 易扭曲"

**右侧：重采样后（Z_ResampleSmooth）**
- 同一条曲线
- 曲线上**等距分布**的 149 个点（均匀小圆点）
- 标注"149 个等弧长点 → 分布均匀"
- 叠一条二次光顺曲线（更深色），标"二次光顺"

### 重采样原理的图示

画一条曲线，标出归一化弧长 0 → 1，在 0%、0.67%、1.34%…100% 处标出等分点：

```
0%        25%        50%        75%       100%
●────●────●────●────●────●────●────●────●   ← 149 个等弧长点（示意取9个）
└───────────── 弧长归一化 0~1 ─────────────┘
第k个点位置 = (k-1)/(149-1)
```

### 该图对应的参数（图旁标注）

| 参数 | 在图上对应 |
|---|---|
| 源样条集 Z_Smooths | 左侧图的来源标注 |
| 重采样点数 149 | 右侧等分点数量标注 |
| 光顺偏差阈值 1.0 | 二次光顺曲线旁标注"最大偏差≤1.0" |
| 点集 Z_ResamplePoints | 指向右侧等分点 |
| 原始样条集 Z_OriginalSpline | 指向右侧重拟合样条 |
| 光顺集 Z_ResampleSmooth | 指向右侧二次光顺曲线 |

> **核心要传达**：重采样让所有截面的点**数量一致、间距均匀**，为 Loft 提供规整输入，避免曲面扭曲。

## 4. 图 ④ 分阶段③ 生成曲面（多曲线 → Loft 蒙皮）

**目的**：讲清"多条截面曲线如何蒙皮成三维叶片曲面"。

### 必须画出的要素（等轴侧/3D 视角）

1. **多条截面曲线沿展向排列**：画 5~7 条截面曲线（椭圆形/翼型轮廓），沿 Z 轴方向平行排列
   - 用虚线表示，标注 `Z_ResampleSmooth`（来自步骤②）
2. **Loft 蒙皮曲面**：在曲线之间画连续的曲面（半透明/网格表示），包络所有曲线
3. **耦合示意**（图示重点）：用几条**纵向引导线**连接相邻截面曲线上的"对应点"，表示 Loft 的耦合方式
   ```
   截面N+1  ╭──●──────●──────●──╮    ← 截面上的等距点（来自②）
            │  │      │      │   │
   引导线    │  │      │      │   │   ← 纵向耦合线（section_coupling）
            │  │      │      │   │
   截面N    ╰──●──────●──────●──╯
   ```
4. **最终曲面**：标 `loft_surface`，标注"叶片三维曲面"

### 该图对应的参数（图旁标注）

| 参数 | 在图上对应 |
|---|---|
| 源曲线集 Z_ResampleSmooth | 指向多条截面曲线 |
| 截面耦合方式 1 | 指向纵向引导线（解释耦合=截面间点的对应规则） |
| 重新限定 1 | 曲面两端（翼根/翼尖端面）的处理标注 |
| 规范检测 2 | 图角注"规范元素检测级别" |

> **核心要传达**：步骤②让每条曲线的点数一致，是**耦合能成功的前提**——否则相邻截面的点对不齐，纵向引导线会错乱，曲面扭曲。

## 5. 四张图的总体要求（画图规范）

1. **少用流程箭头**：除了总成图的主数据流，分阶段图聚焦"几何本身"，不要重复画 ①→②→③ 的箭头
2. **多用几何局部图**：翼型截面、点序号、曲线点分布、3D 蒙皮——这些是讲清机理的关键
3. **点序号必须标**：前缘(200)、尾缘(1/399) 是所有参数的基准，分阶段①图必标
4. **前后对比**：步骤②用"重采样前 vs 后"对比图最直观
5. **配色**：建议沿用工具/文档的蓝色系（主色 #0ea5e9、深钢蓝 #1e3a5f）做强调，点/线用深色，曲面用半透明蓝
6. **参数标注对应**：每张图旁列出"该图对应的参数 → 图中位置"对照表（上文已给出模板）
