# 关于「气动组工具箱」

> AERO TOOLKIT

一个可扩展的桌面工具集，左侧深钢蓝导航栏列出所有工具，右侧白底工作区显示当前选中工具的操作界面。

---

## 版本

见导航栏底部「版本 vX.X」标签，或 `src/config.py` 中的 `APP_VERSION`。

## 已注册模块

| 模块 | 说明 |
|---|---|
| **风场数据统计** | 风场秒级时间序列 / 月度湍流表的批量统计与可视化 |
| **风场对比** | 多个风场的月均风速 / 密度横向对比（输入为各风场的 `风场统计数据.xlsx`） |
| **叶片形状输出** | 三阶段叶片几何输出（AppData + GEO → XFOIL 修正 → 最终几何文件）|

> 加新工具的开发者指南见项目根目录的 `使用说明.md` §8。

## 路径管理

所有模块的输入/输出路径由**全局配置中心**（`src/global_config.py`）统一管理：

- 配置文件：`config\.paths.json`（首次启动自动创建）
- 持久化入口：导航栏底部「⚙ 设置」按钮
- 面板内「浏览」仅会话内临时生效，不持久化
- 首启迁移：若 `config\.paths.json` 不存在，会把 `输入数据\` 顶层的旧文件自动移入 `输入数据\wind_farm\`
