# 分段复用拟合（双端 C2 连续） — 帮助

> 入口：曲线拟合模块 → 「✂ 分段复用 (C2)」Tab

本工具用于在已有叶片几何参数基础上，**保留两侧原值不动**，
只对中段做 B 样条重设计，并在两个分段点处强制 **双端 C2 连续**
（一阶导 + 二阶导都与对应侧的复用段衔接）。

适用于：预弯、后掠、弦长、扭角、相对厚度等任何沿展向的曲线。

---

## 1. 应用场景

| 场景 | 做法 |
|---|---|
| **双段重设**（v0.3.09 起默认）：叶尖中段需要重新光顺 | 设 R1=0.50、R2=0.70，中段 [0.5R, 0.7R] 重设，两侧 [0, 0.5R] 和 [0.7R, 1R] 复用 |
| **共模设计**：内段已锁定（继承自基准叶片），只调叶尖 | 设 R1=0.7、R2=1.00（退化），仅 [0, 0.7R] 复用 + [0.7R, 1R] 重设 |
| 叶尖形状优化（预弯 / 后掠重新分布） | 从 STAGE-1 几何表 Prebend/Sweep 列另存 CSV，再粘贴或载入 |
| 叶尖扭角重新光顺 | 同上，从 Twist 列导出 CSV，分段点设在扭角转折处 |

---

## 2. 数据输入

> **v0.3.11 起**：数据输入方式对齐 Tab 1「全段拟合」—— `载入文件 / 解析 / 清除` 三按钮 + 可编辑文本框。不再依赖 STAGE-1 几何表的多列选择（如需拟合 Prebend/Sweep/Chord 等，请先在 STAGE-1 模块把对应列另存为两列 CSV）。

### 2.1 三步流程

| 步骤 | 按钮 | 行为 |
|---|---|---|
| 1. 选文件 | `📂 载入文件` | 弹文件对话框（txt/csv/dat），文本自动填入编辑框 |
| 2. 解析 | `⚙ 解析` | 第 1 列当 X（自动归一化到 [0, 1] 作为 r/R），第 2 列当 Y |
| 3. 清除 | `🗑 清除` | 清空编辑框 + 数据 + 拟合结果 |

> 也可**直接粘贴**两列数据进编辑框再点解析（空白行 / `#` 开头注释自动跳过）。

### 2.2 数据格式

```
# 展向位置   预弯量
0.000       0.000
0.050      -0.014
0.100      -0.056
...
1.000      -1.400
```

- 第 1 列：展向位置（任意单调数值，解析后自动缩放到 [0, 1]）
- 第 2 列：Y 值（预弯 / 后掠 / 弦长 / 扭角等，单位自定）
- 行内多列只取前两列，其余忽略

## 3. 分段与拟合参数

| 参数 | 默认 | 说明 |
|---|---|---|
| 分段点 R1 (r/R) | 0.50 | 左复用段 [0, R1] 锁定原数据；中段 [R1, R2] 可编辑 |
| 分段点 R2 (r/R) | 0.70 | 右复用段 [R2, 1] 锁定原数据。**R2 = 1.00 时退化为单分段点**（无右段） |
| 连续性 | C2（双端固定） | R1 处 y/y'/y'' 匹配左段末点；R2 处 y/y'/y'' 匹配右段首点（has_right=True 时） |
| B 样条阶数 k | 5 | 五次样条。**双端 C2 要求 k ≥ 5**（左 2 + 右 2 = 4 = k−1） |
| 中段控制点数 | 5 | 不含 R1/R2 端点。变化时旧 Y 自动重采样到新点数 |

> **R1 / R2 顺序约束**：UI 强制 R2 ≥ R1 + 0.05（最小中段宽度 0.05），违反时自动调整。
>
> **C2 的工程含义**：分段点处不仅切线方向连续（C1），曲率也连续（C2），所以叶片表面在该处不会出现肉眼可见的"折线"。双端 C2 让整条曲线在 R1 和 R2 处都无折角。

---

## 4. 编辑中段

两种方式（双向同步）：

### 4.1 画布拖动

直接用鼠标**左键按住红色控制点**拖动 → 中段蓝色拟合曲线实时跟随。

- **X、Y 都可拖**：拖动同时调整控制点的 X 和 Y
- **X 自动约束**：
  - 保持在 (R1, R2) 开区间内
  - 与左右邻居保持严格递增（不会越过邻居）
- 鼠标悬停红点时变为抓取手势，拖动时变为闭合手势

### 4.2 表格编辑

右下方表格显示控制点的 (X, Y)。双击单元格进入编辑：

- **改 Y**：直接输入数值
- **改 X**：需要保证严格递增且在 (R1, R2) 开区间内，否则编辑被忽略（保留旧值）

---

## 5. 分段点状态

画布下方状态条显示两个分段点处的实时值：

```
R1=0.50: y=-1.393, y'=-5.57, y''=-11.14  |  R2=0.70: y=-2.729, y'=-7.80, y''=-11.14
（左/右段端部局部样条估计，作为中段 C2 约束）
```

- `y/y'/y''`：分段点处的原始值与一阶/二阶导
- R1 处的导数：用**左复用段**末尾 ~8 个点拟合局部三次样条求得
- R2 处的导数：用**右复用段**开头 ~8 个点拟合局部三次样条求得
- 退化情形（R2=1.0）：状态条只显示 R1 约束 + "R2=1.00（退化，叶尖自由）"

---

## 6. 输出

点 `💾 导出合并曲线 CSV`：

- 默认文件名：`{列名}_segmented_fit_{时间戳}.csv`（如 `Prebend_segmented_fit_20260624-153012.csv`）
- 默认目录：`输出/curve_fitter/`
- 内容：两列（`span_ratio`, `value`），UTF-8 BOM（Excel 双击直开）
- 行数 = 左段原数据点数 + 中段拟合采样点数 + 右段原数据点数（典型 50–150 行）

> **导出后怎么用**：CSV 拿去其他工具做下游处理，或手工合并回 GEO 文件的对应列。
> 本工具**不直接回写** shape_design 的 STAGE-1 输出（避免破坏原始结果）。

## 7. 常见问题

| 现象 | 原因 / 处理 |
|---|---|
| 提示「左段点数过少」 | R1 调得太低；原始数据在 [0, R1] 内不足 4 个点。把 R1 往大调 |
| 提示「右段点数过少」 | R2 调得太高；原始数据在 [R2, 1] 内不足 4 个点。把 R2 往低调 |
| 提示「中段控制点数少于阶数 k」 | k=7 时控制点至少 7 个。增加控制点数或降低 k |
| 拖动控制点没反应 | 鼠标没点中红点圆心（命中半径 20 像素）；放大画布或更靠近中心点 |
| 改 X 编辑被忽略 | X 必须严格递增且在 (R1, R2) 开区间内。表格会保留旧值不强制回写 |
| 想回到全段拟合 | 切到 Tab 1「📈 全段拟合」（原有功能未改动） |
| 想退化为单分段点 | 把 R2 调到 1.00，右复用段消失，行为与 v0.3.08 之前一致 |
| 想拟合 STAGE-1 几何表的某一列 | 先到 STAGE-1 模块把该列（Prebend/Sweep/Chord 等）另存为两列 CSV（r/R + 该列值），再到本 Tab「📂 载入文件」读取（v0.3.11 起不再直接读 xlsx） |

## 8. 算法原理

```
左段 [0, R1]
   ↓ 取末尾 ~8 个点 → 局部三次样条 → 求 R1 处 y', y''
右段 [R2, 1]   (若 R2 < 1.0)
   ↓ 取开头 ~8 个点 → 局部三次样条 → 求 R2 处 y', y''
中段控制点 (用户拖动)
   ↓
scipy.interpolate.make_interp_spline(k=5, bc_type=(
    [(1, y1'), (2, y1'')],   ← 左端 C2 约束（来自左段）
    [(1, y2'), (2, y2'')]    ← 右端 C2 约束（来自右段；退化时为 [(2,0),(3,0)] 自然边界）
))
   ↓
中段密集采样 → 拼到左段后面 → 拼到右段前面 → 完整曲线
```

> **为什么 k=5（五次样条）？** scipy `make_interp_spline` 要求边界约束总数 = k−1。
> 双端 C2 每端 2 个约束 = 总 4 个，故 k−1 = 4 → k = 5。
> 若只用单端 C2（退化情形 R2=1.0），k=3 也可，但 UI 统一用 k=5 并在右端补
> "自然边界"零导数约束 [(2, 0), (3, 0)]，行为稳定一致。

> **边界连续性通过 `bc_type` 显式传给 scipy，不需要整体联立求解，稳定且快。**
