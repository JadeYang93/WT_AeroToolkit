# 3D 造型 · 步骤② 重采样光顺

> 对应面板 Tab「② 重采样光顺」，业务函数 `resample_and_smooth()`
> 对应参考脚本 `1227-2opt.py`

## 这个步骤做什么

把步骤①输出的每条**光顺截面曲线**，按等弧长**重新采样成均匀分布的点**，再用这些点重新拟合样条并做二次光顺。

一句话：**把截面曲线的点重排成均匀分布，再做一次光顺，提升后续曲面质量**。

## 为什么需要这一步

步骤①的样条是由原始点云直接拟合的，点往往**分布不均**（前缘密、中段稀，或某段点缺失），导致：
- 样条在稀疏区可能出现波动
- 多截面曲面（步骤③）时，不同截面点数/分布不一致，曲面易扭曲

重采样让每条曲线的点**数量一致、间距均匀**，为多截面曲面提供规整的输入。

## 前置条件

- 已运行步骤①，CATIA 文档里存在 `Z_Smooths` 几何集（含多条光顺曲线）

## 输入

| 项 | 说明 |
|---|---|
| 源样条集 `source_set` | 从哪个几何集读样条（默认 `Z_Smooths`，承接步骤①） |
| 重采样点数 `num_points` | 每条曲线重采样成多少个点 |

## 处理逻辑（逐曲线）

对源几何集里的每条曲线：

1. **等弧长重采样**：沿曲线按 0 → 1 的归一化弧长，均分 `num_points` 个点
   - 第 k 个点位置 = `(k-1) / (num_points-1)`，k = 1..num_points
   - 例：num_points=149，则在曲线 0%、0.67%、1.34%、…、100% 处各取一点
2. **重拟合原始样条**：用这 num_points 个点重新连成一条样条（作为对比基准，存入 `Z_OriginalSpline`）
3. **二次光顺**：对重拟合的样条再做光顺（`AddNewSplineSmooth`），按偏差阈值控制

## 输出（写入 CATIA 几何集）

| 几何集名 | 内容 |
|---|---|
| `Z_ResamplePoints` | 每条曲线重采样后的等距点集 |
| `Z_OriginalSpline` | 重采样点重新拟合的样条（未光顺） |
| `Z_ResampleSmooth` | 二次光顺后的样条（**喂给步骤③**） |

> ⚠ 几何集名可在「高级参数」栏自定义。

## 参数详解

### 核心参数

| 参数 | 默认 | 含义 |
|---|---|---|
| 源样条集 `source_set` | `Z_Smooths` | 从哪个几何集读取待处理的样条 |
| 重采样点数 `num_points` | 149 | 每条曲线重采样成多少个等距点。点数越多，曲线细节保留越充分，但点数过多会降低光顺效果；点数太少会丢失翼型特征 |
| 光顺偏差阈值 `smooth_max_deviation` | 1.0 | 二次光顺允许的最大偏差。值越大光顺越强（偏离原始曲线越多），值越小越贴合原始曲线 |

### 高级参数

| 参数 | 默认 | 含义 |
|---|---|---|
| 相切阈值 `tangency_threshold` | 0.5 | 光顺时的相切连续性阈值 |
| 校正模式 `correction_mode` | 3 | 光顺校正算法模式 |
| 点集 / 原始样条集 / 光顺集 | `Z_ResamplePoints` / `Z_OriginalSpline` / `Z_ResampleSmooth` | 各类输出的几何集名 |

## 运行结果

- 进度日志逐曲线输出：`✓ 曲线 i/N: 重采样 149 点 + 光顺`
- 失败曲线会跳过并记录
- 完成后弹窗汇总：`重采样光顺完成: X 条曲线`

## 常见问题

**Q: 提示"几何集不存在: Z_Smooths"？**
A: 还没运行步骤①，或步骤①的光顺集名被改过。先跑步骤①，或在「源样条集」填入实际存在的几何集名。

**Q: 重采样后曲线比原来"平"了，丢了细节？**
A: 重采样点数太少（丢失高频特征）或光顺偏差阈值太大（过度光顺）。试着增大 `num_points`（如 200）、减小 `smooth_max_deviation`（如 0.5）。

**Q: 想单独处理手动导入的样条（不经过步骤①）？**
A: 把手动样条放进一个几何集（如 `line`），在「源样条集」填该集名即可单独跑本步骤。
