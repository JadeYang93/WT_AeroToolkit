# 曲线拟合使用说明

> 路径：工具箱 → 曲线拟合
> 模块 ID：`curve_fitter`

## 这是什么

把一组离散的截面数据（如展向坐标 + 翼型几何量）拟合成光滑曲线，
再按需在任意 X 位置插值。源项目 `F:/python/叶片前后缘光顺/pyqt_curve_fitter11.py`
（单文件 2089 行）已重写为工具箱风格：业务逻辑在 `src/curve_fitter/`，
UI 在 `src/tools/curve_fitter_panel.py`。

> 预弯度相关功能已剥离为独立模块「📐 预弯设计」（`prebend_design`）。

## 数据输入

- **第 1 列为 X**，其余列为 Y1, Y2, ... Yn（至少 2 列）
- 支持空行 / `#` 开头的注释行
- 空白（空格 / Tab）、英文逗号千位分隔符都兼容
- 两种方式：
  - 粘贴到左侧「数据」文本框 → 点「⚙ 解析」
  - 点「📂 载入文件」加载 `.txt` / `.csv` / `.dat`（多编码回退：utf-8 / latin-1 / utf-16 / gbk / cp936 / gb2312 / big5 / chardet）

最大数据量 `MAX_ROWS = 10000` 行。

## 拟合方法

下拉框切换，方法名后括号内是 `curve_fitter` 内部 key：

| 显示名 | 内部 key | 算法 | 备注 |
|---|---|---|---|
| B 样条 | `spline` | `scipy.interpolate.UnivariateSpline` | 支持 smooth 平滑因子 |
| 三次样条 | `cubic` | `scipy.interpolate.CubicSpline` | 过所有点 |
| Akima | `akima` | `scipy.interpolate.Akima1DInterpolator` | 抗振荡 |
| PCHIP | `pchip` | `scipy.interpolate.PchipInterpolator` | 保形无过冲 |
| 多项式 | `poly` | `numpy.polyfit + poly1d` | 需指定阶数（1–12，默认 3） |

**参数**：
- `平滑因子 s`：仅 B 样条启用；`s=0` 精确插值，`s>0` 平滑度递增
- `多项式阶数`：仅多项式启用；推荐 ≤ 6，过高会过冲（Runge 现象）

**显示列下拉**：默认「全部 Y 列」，也可选某一列单独查看 / 导出。

## 操作流程

1. 载入 / 粘贴数据 → 点「⚙ 解析」（弹出列数 / 行数确认）
2. 选方法 → 调 smooth / 阶数 → 点「▶ 生成拟合曲线」
3. 画布显示：原始散点（圆点）+ 拟合曲线（实线），按 Y 列分配颜色
4. （可选）填「插值 X」（每行一个）→ 点「计算插值」→ 右下表格显示 X / Yn
5. （可选）勾选「在图上显示插值点」→ 插值点以星形标记叠加
6. 保存：「💾 拟合曲线」/「💾 插值结果」→ 选择路径导出 CSV

## 输出

| 操作 | 默认文件名 | 列 |
|---|---|---|
| 拟合曲线 | `fitted.csv` | `X, Y1, Y2, ...`（1000 个等距采样点） |
| 插值结果 | `interp.csv` | `X, Y1, Y2, ...`（用户输入的 X） |

> 默认保存到 `输出/curve_fitter/`，可在保存对话框里改任意路径。

## 常见问题

- **拟合曲线振荡**：换 Akima / PCHIP；或调小 smooth；多项式阶数别超 6
- **插值结果出现 N/A**：目标 X 超出原始数据范围，对应方法不支持外推（如 CubicSpline）
- **B 样条 smooth 怎么选**：`s=0` 过所有点（适合精确数据），`s = len(x)` 量级开始明显平滑
- **多项式阶数 vs 点数**：阶数必须 < 点数，否则 `polyfit` 报错；推荐 点数 ≥ 阶数 + 2
- **多列 Y 颜色循环**：蓝/红/绿/品红/青/黄/黑/橙/紫/棕，超过 10 列循环

## 路径配置

输入 / 输出目录在「⚙ 设置」里改。输入文件可通过「📂 载入文件」按钮任意选，
不强制放在 `输入数据/curve_fitter/` 下。
