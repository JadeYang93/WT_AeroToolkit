# 风场对比 — 帮助

> 模块 ID：`wind_farm_compare`　　默认输入：`输入数据\wind_farm_compare\`　　默认输出：`输出\wind_farm_compare\`

本模块把多个风场的月均风速 / 密度拉到一张图上横向对比。数据来源是 **风场数据统计** 模块已经生成的 `风场统计数据.xlsx`，每个风场一个子目录。

---

## 1. 数据准备

每个风场 = 输入目录下的一个**子目录**，子目录内必须包含 `风场统计数据.xlsx`。

```
输入数据\wind_farm_compare\
├── 风场A\
│   └── 风场统计数据.xlsx      ← 从「风场数据统计」模块的输出目录拷过来
├── 风场B\
│   └── 风场统计数据.xlsx
└── 风场C\
    └── 风场统计数据.xlsx
```

**子目录名 = 风场名**，会作为图例和 Excel 列名直接显示。

> **重要**：`风场统计数据.xlsx` 必须由「风场数据统计」模块在勾选了**「月」粒度**的情况下生成，否则会缺 `月均风速` / `月均密度` sheet，导致本模块读到空数据。

---

## 2. 输入约定

| 项目 | 要求 |
|---|---|
| 子目录 | 每个风场一个；隐藏目录（`.` 开头）会被忽略 |
| 子目录内文件 | `风场统计数据.xlsx`（文件名严格匹配） |
| Sheet 名 | `月均风速` / `月均密度`（也兼容 `月平均风速` / `月平均密度`） |
| Sheet 列 | `机组号` / `月份` / `月平均风速`（或 `月平均密度`）/ `数据天数` |
| 月份格式 | `YYYY-MM` 字符串（如 `2024-05`） |

> 缺 `风场统计数据.xlsx` 的子目录会被跳过，日志提示 `[警告] 风场 X 缺 风场统计数据.xlsx`。

---

## 3. 聚合方式

**风场整体**对比（不是按机组）：

1. 取该风场 `月均风速` sheet 中所有机组的月均
2. 按月份 `groupby` 后简单平均，得到「该风场该月的全风场均值」
3. 每个风场一条线

> 例如：风场 A 有 12 台机组，2024-05 的月均风速 = 12 台机组该月月均风速的算术平均。

---

## 4. 面板操作

### 4.1 数据源

- **路径**：当前输入目录（来自全局配置中心，初始为 `输入数据\wind_farm_compare\`）
- **扫描**：扫描子目录，显示「找到 N 个风场（M 个就绪）」+ 风场名列表
- **浏览...**：选其他文件夹。**仅本次会话生效**，永久修改请用「⚙ 设置」

### 4.2 参数

- **对比指标**：风速 / 密度（多选，至少 1 个）
- **日期范围**：
  - 勾选 **全部日期**：用扫描到的全部月份（默认）
  - 取消勾选：手动选择起止月份

### 4.3 开始统计

点 **开始统计** 后台线程运行。完成后日志显示生成图数与 Excel 路径，**打开输出** 按钮启用。

---

## 5. 输出结果

输出落在模块输出目录（默认 `输出\wind_farm_compare\`），运行时自动创建。

### 5.1 对比图（PNG）

| 指标 | 文件名 |
|---|---|
| 风速 | `风场月均风速对比.png` |
| 密度 | `风场月均密度对比.png` |

- 横轴 = 月份（并集，自动按时间排序）
- 每风场一条线，颜色用 `_distinct_colors` 自动分配（≤7 个走品牌色，>7 走 tab20 高对比）
- 数据点上方标注数值（风场数 > 12 时省略避免重叠）
- 右上角汇总表：每风场的全期均值

> 风场间月份范围不一致（如风场 C 只有 5-12 月数据）时，缺失月份留空，不补 0。

### 5.2 汇总 Excel

`风场对比汇总.xlsx`，每个指标一个 sheet：

| sheet 名 | 结构 |
|---|---|
| `月均风速对比` | 行=月份，列=风场名，值=该风场该月均值 |
| `月均密度对比` | 同上 |

---

## 6. 常见问题

| 现象 | 处理 |
|---|---|
| 扫描显示「未找到任何子目录」 | 在输入目录下为每个风场建一个子目录，放入 `风场统计数据.xlsx` |
| 扫描显示「缺 Excel: 风场X」 | 该子目录缺 `风场统计数据.xlsx`；从「风场数据统计」模块输出拷过来 |
| 日志提示「缺 月均风速 sheet」 | 在「风场数据统计」模块跑该风场时未勾选「月」粒度，重跑即可 |
| 图里某风场只有部分月份 | 该风场数据本身覆盖范围短（如 5 月才有），符合预期 |
| 某风场整条线没出现 | 该风场所有月份均 NaN（清洗后无有效数据），去原模块检查原始数据 |
| 修改了路径但重启后丢失 | 面板内「浏览」仅会话内生效，永久修改请用「⚙ 设置」 |
