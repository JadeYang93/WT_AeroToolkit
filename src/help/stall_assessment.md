# 失速评估使用说明

> 路径：工具箱 → 失速评估
> 模块 ID：`stall_assessment`

## 这是什么

由「相对厚度 → 失速攻角」的标准翼型表，求出叶片沿展向各站的失速攻角分布；
若叶片部分展向段装了 VG（涡发生器），可按子段指定 VG 状态，对应厚度端点
改用 VG 版失速攻角插值。

标准翼型表给出了几个离散翼型的相对厚度和对应的失速攻角；叶片的展向分布给出了每个
展向位置的相对厚度。本模块把每个展向位置的厚度代入插值曲线，得到该位置的失速攻角。

## 失速评估 SOP（标准操作流程）

完成一次失速评估的整体流程，前 5 步在 Bladed 中准备"最大攻角沿展向分布"，
第 6 步回到本模块完成插值与对比。点面板上的「📖 失速评估 SOP」可随时查看。

| # | 操作 | 说明 |
|---|---|---|
| 1 | **pcoeffs 计算** | 在 Bladed 中运行 pcoeffs 计算，得到各风速下的功率系数。 |
| 2 | **control k_opt 更新** | 更新最优控制增益 k_opt（变量桨距增益），保证后续稳态计算的桨距角跟踪最优。 |
| 3 | **steadyop → Calculation Output → Aerodynamic Information → First Blade** | 在 steadyop（稳态计算）中，将 Calculation Output 的 Aerodynamic Information 输出项设为 First Blade，确保输出第一片叶片的气动信息。 |
| 4 | **data view → steadyop → Summary Info → Electrical Power → 找额定功率对应风速 − 0.1** | 在 Data View 中查看 steadyop 的 Summary Info，定位 Electrical Power 等于额定功率对应的风速；该风速减 0.1 m/s 通常就是攻角最大的风速（可在 ±0.1° 桨距角下对比攻角以验证）。 |
| 5 | **steadyop → Blade 1 Information → Angle of Attack → 对应风速下的攻角分布** | 在 steadyop 的 Blade 1 Information 中取 Angle of Attack，导出"上一步定位的风速"对应的沿展向攻角分布。 |
| 6 | **标准翼型失速攻角 + 相对厚度插值 + 与最大攻角对比（本模块）** | 将"标准翼型表（相对厚度 → 失速攻角）"填入左上表，配入展向厚度分布（中部）与第 5 步得到的攻角分布，点击「计算失速攻角」——本模块自动按相对厚度插值求出各站失速攻角，并与最大攻角对比、标注失速起始位置。 |

## 输入数据格式

四组数据，都在面板里直接给出（无需文件）：

**1. 标准翼型表（左上表格，可增删行）**

三列：

| 相对厚度 | 标失速攻角 (°) | VG 失速攻角 (°, 可选) |
|---|---|---|
| 18 | 15.5 | — |
| 25 | 12.0 | — |
| 30 | 13.0 | 15.0 |
| 40 | 10.0 | 12.0 |
| 100 | 0.0 | — |
| ... | ... | ... |

- 至少需要 2 行才能插值。
- **VG 失速攻角** 列：填 = 该厚度翼型有 VG 变体；空 = 无 VG 变体。
  VG 失速攻角通常比标失速攻角大（VG 推迟失速）。
- 用「+ 添加 / − 删除」增删行；删除时选中某行则删该行，否则删最后一行。

**2. 展向分布（中部文本框，或载入文件）**

每行「展向位置, 相对厚度」，首行可带标题（自动跳过）。列分隔符兼容逗号 / 空格 / 制表符：

```
r/R, t/c
0.00, 1.00
0.20, 0.60
0.50, 0.30
1.00, 0.18
```

- 可直接粘贴文本；
- 或点「📂 载入 CSV/xlsx」从文件读（取前两列，首行可带标题）。

**3. 攻角分布（中部文本框，或载入文件）**

每行「展向位置, 最大攻角°」，格式同上。用于与失速攻角对比，求交点（失速起始位置）。

> **单位约定**：标准表和展向分布里的相对厚度单位必须一致（同用小数 0.18，或同用百分数 18），
> 模块不做单位换算。

**4. VG 安装范围表（底部表格，可选）**

每行描述「哪个标准厚度的翼型在哪个 z 子段内装了 VG」：

| 标准厚度 | VG 起 z (r/R) | VG 止 z (r/R) |
|---|---|---|
| 40 | 0.10 | 0.25 |
| 30 | 0.20 | 0.35 |

- **标准厚度** 列：下拉菜单，选项 = 标准翼型表里 VG 失速攻角列非空的厚度。
- **VG 起 / 止 z**：用户填，叶片上 VG 安装段的起 / 止展向位置（无量纲 r/R）。
- 同一标准厚度可以有多行（多段 VG，理论上不重叠）。
- **表为空** = 全展向无 VG → 退化为 PCHIP 兼容路径（与无 VG 版完全一致）。

## 插值方法

### 默认路径（无 VG）

PCHIP（分段三次 Hermite 保形插值，`scipy.interpolate.PchipInterpolator`）。

特点：单调保形，不会在标准点之间产生振荡或过冲——这是失速攻角这类物理量插值的首选，
避免插出超出合理范围的攻角值。

### VG 路径（VG 安装表非空）

对每个展向位置 z：

1. 找当前相对厚度 t(z) 落在哪两个相邻标准厚度之间（区间 [t_a, t_b]，t_a > t_b）。
2. 分别决定 t_a、t_b 在 z 处用标攻角还是 VG 攻角：z 在该厚度的 VG 安装段内 → α_VG，
   否则 → α_标。
3. 在 [t_a, t_b] 内对当前 t(z) 做两点线性插值：
   α = α_a + (α_b − α_a) × (t_a − t(z)) / (t_a − t_b)

**为什么 VG 段用两点线性而不是 PCHIP？** VG 的作用只是在某些 z 子段内把某些标准
厚度点的攻角值替换为 VG 版；端点对变化后，PCHIP 的全局形状会"涟漪"扩散，不符合
"VG 只影响局部"的物理直觉。两点线性保证 VG 影响严格限制在涉及该厚度端点的子段内。

**连续性**：

- 段内（同一对端点）：两点线性，必然连续。
- 段边界（不同厚度区间交接）：端点共享相同厚度和攻角，自动连续。
- VG 装卸边界（同厚度区间内 z 子段切换 VG 状态）：端点对从 (α_std, α_VG) 切换为
  (α_std, α_std)，**攻角会跳变**——这是 VG 终止的物理表现，符合预期。

**超范围处理**：若展向分布里有相对厚度超出标准表的范围（常见于叶根，如叶根厚度 1.0
而标准表只到 0.6），会**报错提示补全标准表**，不做外推（避免失真）。

## 输出

全部写入 `输出/stall_assessment/`：

| 文件 | 内容 |
|---|---|
| `stall_alpha_span.csv` | 无 VG：三列 `span_position, relative_thickness, stall_alpha_deg`；<br>有 VG：追加 `vg_active` 列（1.0/0.0，该 z 是否落在任何 VG 段内） |

面板内同时显示：
- **失速攻角展向分布图**（右）：展向位置 r/R ↔ 失速攻角
  - VG 段：橙色阴影 + 灰色竖虚线（VG 边界）
  - 失速攻角曲线与最大攻角曲线对比，标注交点（失速起始位置）
- **结果表**（右上）：可复制

## UI 操作

1. 左上表格填好 / 增删标准翼型点（相对厚度 + 标失速攻角 + VG 失速攻角）
2. 中部粘贴展向分布 + 攻角分布，或「📂 载入 CSV/xlsx」
3. （可选）底部 VG 安装范围表：填 VG 段厚度（下拉）+ 起/止 z
4. 点「▶ 计算失速攻角」→ 后台线程执行插值 + 写 CSV
5. 完成后右侧出图、结果表填充，「📋 复制结果」可拷贝到剪贴板
6. 「📂 打开输出目录」直接打开 `输出/stall_assessment/`

## 路径配置

输出目录在「⚙ 设置」里改。改完即时生效，无需重启。
