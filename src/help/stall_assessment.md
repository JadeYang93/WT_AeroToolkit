# 失速评估使用说明

> 路径：工具箱 → 失速评估
> 模块 ID：`stall_assessment`

## 这是什么

由「相对厚度 → 失速攻角」的标准翼型表，用 PCHIP 保形插值，求出叶片沿展向各站的失速攻角分布。

标准翼型表给出了几个离散翼型的相对厚度和对应的失速攻角；叶片的展向分布给出了每个
展向位置的相对厚度。本模块把每个展向位置的厚度代入插值曲线，得到该位置的失速攻角。

## 输入数据格式

两组数据，都在面板里直接给出（无需文件）：

**1. 标准翼型表（左上表格，可增删行）**

两列：

| 相对厚度 | 失速攻角 (°) |
|---|---|
| 0.18 | 12.5 |
| 0.25 | 14.5 |
| ... | ... |

- 至少需要 2 行才能插值。
- 用「+ 添加 / − 删除」增删行；删除时选中某行则删该行，否则删最后一行。

**2. 展向分布（左下文本框，或载入文件）**

每行「展向位置, 相对厚度」，首行可带标题（自动跳过）。列分隔符兼容逗号 / 空格 / 制表符：

```
r/R, t/c
0.00, 1.00
0.20, 0.60
0.50, 0.30
1.00, 0.18
```

- 可直接粘贴文本；
- 或点「📂 载入 CSV/xlsx」从文件读（取前两列，首行可带标题）。

> **单位约定**：标准表和展向分布里的相对厚度单位必须一致（同用小数 0.18，或同用百分数 18），
> 模块不做单位换算。

## 插值方法

PCHIP（分段三次 Hermite 保形插值，`scipy.interpolate.PchipInterpolator`）。

特点：单调保形，不会在标准点之间产生振荡或过冲——这是失速攻角这类物理量插值的首选，
避免插出超出合理范围的攻角值。

**超范围处理**：若展向分布里有相对厚度超出标准表的范围（常见于叶根，如叶根厚度 1.0
而标准表只到 0.6），会**报错提示补全标准表**，不做外推（避免失真）。

## 输出

全部写入 `输出/stall_assessment/`：

| 文件 | 内容 |
|---|---|
| `stall_alpha_span.csv` | 三列：`span_position, relative_thickness, stall_alpha_deg` |

面板内同时显示：
- **插值校核图**（右上）：标准翼型点（散点）+ PCHIP 插值曲线，直观判断拟合是否合理
- **失速攻角展向分布图**（右下）：展向位置 r/R ↔ 失速攻角
- **结果表**（右下）：可复制

## UI 操作

1. 左上表格填好 / 增删标准翼型点（相对厚度 + 失速攻角）
2. 左下粘贴展向分布，或「📂 载入 CSV/xlsx」
3. 点「▶ 计算失速攻角」→ 后台线程执行插值 + 写 CSV
4. 完成后右侧出图、结果表填充，「📋 复制结果」可拷贝到剪贴板
5. 「📂 打开输出目录」直接打开 `输出/stall_assessment/`

## 路径配置

输出目录在「⚙ 设置」里改。改完即时生效，无需重启。
