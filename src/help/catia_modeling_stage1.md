# 3D 造型 · 步骤① 构建截面

> 对应面板 Tab「① 构建截面」，业务函数 `build_sections()`
> 对应参考脚本 `Pre-process.py`

## 这个步骤做什么

把 CATIA 文档里**按截面分组的点云**，逐截面（Sect 组）自动构建出叶片一个截面的完整几何要素：**拟合样条 → 光顺曲线 → 参考平面 → 前缘点 / 尾缘点 → 弦线**。

一句话：**从散乱点云，重建出每个截面的翼型轮廓几何**。

## 前置条件

1. CATIA 已启动，并打开一个**零件文档（.CATPart）**
2. 文档中已导入点云，且点按 `Sect{组号}_{点序号}` 命名（如 `Sect1_1`、`Sect1_2`、…、`Sect1_399`、`Sect2_1`、…）
   - 命名约定来自「叶片形状输出」STAGE-3 导出的 `3D_points.stp`
   - 一个 `Sect{组号}` 对应叶片的一个截面（如翼根到翼尖的某个站位）

## 输入

| 项 | 说明 |
|---|---|
| 点云 | CATIA 文档内已有的点，按 `Sect{组}_{点}` 命名 |
| 截面数 `num_groups` | 要处理的截面组数（从 `start_group` 起） |
| 每截面点数上限 `points_per_section` | 每个截面最多取多少个点（点序号 1 到此上限） |

## 处理逻辑（逐截面）

对每个组号 `g`（`start_group` 到 `start_group + num_groups - 1`）：

1. **收集点**：按 `Sect{g}_1`、`Sect{g}_2`、… 依次查找点，直到点序号达上限或找不到点为止
2. **拟合样条**：把这些点用一条 Spline 串起来（`AddNewSpline`，通过点模式）
3. **光顺**：对样条做光顺处理（`AddNewSplineSmooth`），按四段阈值控制光顺强度
4. **参考平面**：以前缘点为基准构造截面平面（用于后续曲率检查 / 弦线测量）
5. **前缘点 / 尾缘点**：按指定点序号提取前缘点（LE）和两个尾缘点（TE）
6. **弦线**：用两个尾缘点连成直线，即为该截面的弦线

## 输出（写入 CATIA 几何集）

步骤运行后，会在 CATIA 零件里新建以下几何集（HybridBody）：

| 几何集名 | 内容 |
|---|---|
| `Z_Splines` | 每个截面的原始拟合样条 |
| `Z_Smooths` | 每个截面的光顺后曲线（**喂给步骤②**） |
| `Z_Planes` | 每个截面的参考平面 |
| `Z_Edges` | 前缘点 + 弦线 |
| `Z_TrailingEdges` | 尾缘点 |

> ⚠ 几何集名可在「高级参数」栏自定义。若改名，后续步骤的源集名也要对应修改，否则串联会断。

## 参数详解

### 核心参数

| 参数 | 默认 | 含义 |
|---|---|---|
| 截面数 `num_groups` | 96 | 处理多少个截面组（从起始组号起算） |
| 起始组号 `start_group` | 1 | 从第几个 Sect 组开始（用于只处理某区段） |
| 四段光顺阈值 | (4, 3, 2, 1) | 样条光顺的四段强度阈值。把样条按长度四等分，各段用对应阈值：值越大光顺越强。四段对应前段/中前/中后/后段，允许对不同区域施加不同光顺强度 |

### 高级参数

| 参数 | 默认 | 含义 |
|---|---|---|
| 点数上限 `points_per_section` | 400 | 每截面最多取的点数（点序号遍历上限） |
| 前缘点序号 `le_point_num` | 200 | 取作前缘点的点序号（在该截面的点序列中第几个） |
| 尾缘点1 `te_point1_num` | 1 | 取作尾缘第一个点的序号 |
| 尾缘点2 `te_point399_num` | 399 | 取作尾缘第二个点的序号（与点1连成弦线） |
| 相切阈值 `tangency_threshold` | 0.5 | 光顺时的相切连续性阈值 |
| 校正模式 `correction_mode` | 3 | 光顺校正算法模式 |
| 样条集 / 光顺集 / 平面集 / 边缘集 / 尾缘集 | Z_Splines / Z_Smooths / Z_Planes / Z_Edges / Z_TrailingEdges | 各类几何输出的几何集名 |

## 运行结果

- 进度日志会逐截面输出：`✓ Sect{组号}: 样条(N点) + 光顺 + 平面 + 前尾缘`
- 若某个截面无点或处理失败，会跳过并记录：`⚠ Sect{组号}: 无点，跳过` / `✗ Sect{组号}: 失败 - ...`
- 完成后弹窗汇总：`构建截面完成: X/Y 成功`

## 常见问题

**Q: 提示某个截面"无点，跳过"？**
A: 该组号在文档里找不到任何 `Sect{组号}_*` 的点。检查点云命名是否完整、起始组号是否正确。

**Q: 光顺后曲线扭曲？**
A: 调小四段光顺阈值（如改成 (2,1,1,0.5)），或检查该截面的点是否按真实翼型顺序排列（点序号应沿翼型轮廓连续）。

**Q: 前缘/尾缘点位置不对？**
A: 前缘点序号、尾缘点序号取错了。默认假设每截面 400 个点、前缘在第 200 点、尾缘在第 1 和第 399 点。若你的点云分布不同，在高级参数里改这三个序号。
